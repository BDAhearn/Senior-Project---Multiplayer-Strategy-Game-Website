## User Account
When signed out, the user is greated with a generic new user Welcome page. They can then go to the User page and select Register or Login. This user already had an account and was able to login. After they logged in, they were directed to the welcome screen with their name displayed. This login is encrypted and secured. It uses a cookie based approach to allow the user to stay logged in for 2 weeks.

## Game
The user can proceed to the Game page for a selection of games (right now just one) and play that game with another person using the same computer. The user does not need to log in to do this. They can choose to play the game over as many times as they want.
